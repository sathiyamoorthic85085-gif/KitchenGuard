# KitchenGuard

## Build

```bash
npm install
npm run build
```

## Preview (stable port)

Use this command to guarantee preview runs on **port 4173**:

```bash
npm run preview:ready
```

If build already exists, you can run only preview:

```bash
npm run preview:host
```

Preview URLs:

- Local: `http://localhost:4173`
- Network: `http://<your-machine-ip>:4173`

## Dev server (stable port)

```bash
npm run dev:host
```

Dev URL:

- `http://localhost:4173`
